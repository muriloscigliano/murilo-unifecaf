# Imagens Full Width

As seguintes imagens devem ser adicionadas na pasta `public/`:

1. **challenge.png** - Imagem para a seção "The Challenge"
   - Caminho: `/challenge.png`
   - Usada após a seção "About"

2. **placeholder-hero.jpg** - Imagem após Hero Section
   - Caminho: `/placeholder-hero.jpg`
   - Usada após a seção Hero

3. **placeholder-architecture.jpg** - Imagem para Architecture Section
   - Caminho: `/placeholder-architecture.jpg`
   - Usada após a seção "Stats"

4. **placeholder-technologies.jpg** - Imagem para Technologies Section
   - Caminho: `/placeholder-technologies.jpg`
   - Usada após a seção "Architecture"

5. **placeholder-prompts.jpg** - Imagem para Prompt Engineering Section
   - Caminho: `/placeholder-prompts.jpg`
   - Usada antes da seção "Full Prompts"

## Especificações

- Todas as imagens são exibidas em **full width** (largura total)
- Border radius: **36px** (arredondado)
- Formato recomendado: JPG ou PNG
- Otimização: Recomenda-se comprimir as imagens para melhor performance

## Como adicionar

1. Coloque as imagens na pasta `public/`
2. Use os nomes de arquivo especificados acima
3. As imagens serão automaticamente servidas em `/nome-do-arquivo.extensao`
